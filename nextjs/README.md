# Next.js Production Boilerplate

This repository is an adapted version of https://github.com/vercel/nextjs-subscription-payments 

## Demo

- https://nextjs.ServerData.com/

## Getting Started

1. [Install node](https://nodejs.org/en/download)
2. In your terminal, run the following commands:

```bash
git clone https://github.com/ServerData/startup-boilerplate.git YOUR_APP_NAME
cd YOUR_APP_NAME
```
3. Use `.env` file from ServerData
```
cd nextjs
cp .env.example .env
```
4. Run the local development server
```
npm install
npm run dev
```

## Stack

- Next.JS (App Router)
- Typescript
- Tailwind
